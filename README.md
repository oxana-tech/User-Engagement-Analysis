# User-Engagement-Analysis
 An interactive dashboard for monitoring and analysing digital product web analytics for  2022-2024. The project includes visualisation of user behaviour, demographic data, and activity trends to support business decision-making.
  Project Idea: 
Create a centralized monitoring system to track key engagement metrics, demographic patterns, and growth trends. The goal is to provide stakeholders with actionable insights on user behavior, platform performance, and growth trajectory to optimize UX and business outcomes.
               
My Tasks:
Collect and structure web analytics data from multiple sources.
Design and build interactive visualizations for language preferences and device usage patterns.
Create time-series analysis for WAU and DAU.Implement trend analysis with forecasting using statistical modeling.Develop a user-friendly dashboard interface for stakeholder presentations.
                
Key Findings:
Ukrainian language users dominate (73% of the base, 159 out of 218 users).
Mobile devices account for 80% of all orders, indicating mobile-first behavior.
WAU shows consistent growth from 2022 to 2024 (correlation R² = 0.974).
Seasonal patterns in activity are visible with an overall upward trend.
                 
Business Value:
Mobile optimization priority: Focus development resources on mobile experience for 80% of users.
Localization strategy: Prioritize Ukrainian language content and support for the primary audience.
Growth forecasting: Reliable trend data enables accurate resource planning and capacity management.
Performance monitoring: Real-time tracking of engagement helps identify retention issues and growth opportunities.
